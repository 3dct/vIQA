viqa.load_utils.ImageArray¶

class viqa.load_utils.ImageArray(input_array)[source]¶

Class for image arrays.

describe(path: str | PathLike | None = None, filename: str | None = None) → dict[source]¶

Export image statistics to a csv file.

Parameters:

  • path (str or os.PathLike, optional) – Path to the directory where the csv file should be saved

  • filename (str, optional) – Name of the csv file

Returns:

stats – Dictionary containing the image statistics

Return type:

dict

Warns:

RuntimeWarning – If no path or filename is provided. Statistics are not exported.

Examples

>>> import numpy as np
>>> from viqa import ImageArray
>>> img = np.random.rand(128, 128)
>>> img = ImageArray(img)
>>> img.describe(path="path/to", filename="image_statistics")
